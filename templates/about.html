About IPScanAPI

IPScanAPI is a simple and efficient IP lookup tool built using Flask. It allows users to fetch real-time details of an IP address or domain, including location, region, and country details. This project is designed for cybersecurity professionals, developers, and researchers who need quick insights into IP addresses.

🔹 Features

  • 🌍 Get detailed IP information (City, Region, Country, Location)
  • 🔄 Works with both IP addresses and domains
  • ⚡ Fast and lightweight, built with Flask
  • 🎨 Simple UI for easy usage
